Export & Upload ke GitHub

1

Edit & Simpan

Edit semua pengaturan di tab Customize, Carousel, Gallery, dll. Klik tombol simpan.

2

Export data.json

Klik tombol di bawah untuk download file data.json yang berisi semua pengaturan.

3

Upload Gambar

Upload gambar ke folder assets/ di repository GitHub (carousel/, gallery/, brochure/, logo/).

4

Commit & Push

Upload data.json ke root repository. Website otomatis update dalam 1-5 menit!
